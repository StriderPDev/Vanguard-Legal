Vanguard – Privacy Policy

Last updated: 24 January 2026

1. Scope and Purpose

This Privacy Policy describes how Vanguard (“the bot”) processes data while operating in Discord servers. Vanguard is a moderation and security bot and processes data only for the purpose of server moderation, abuse prevention, and security.

Vanguard complies with Discord’s Terms of Service, Discord’s Developer Policy, and applicable data protection principles.

2. Data Minimisation

Vanguard follows a strict data minimisation policy. Only the minimum data required to perform moderation and security functions is processed.

3. Data That May Be Processed

  • Discord user IDs
  • Discord server (guild) IDs
  • Channel IDs
  • Message timestamps
  • Message content only when necessary for moderation enforcement
  • Moderation metadata (warnings, strikes, timeouts, raid state)

Vanguard does not collect real names, email addresses, IP addresses, or any data outside of what is provided by Discord’s API.

4. Message Content and Privacy

Vanguard does not monitor or store messages indiscriminately. Message content is evaluated only when required to:

  • Detect spam, abuse, or rule violations
  • Enforce server-configured moderation rules

Private messages between users are never read or accessed.

5. Direct Messages From the Bot

Vanguard may send direct messages to users for moderation-related notifications (such as warnings or timeouts). These messages are informational only.

6. Data Storage and Retention

By default, Vanguard stores moderation data in memory only. Most data is temporary and is cleared automatically when the bot restarts.

If persistent storage is implemented in the future, it will be limited to moderation history and configuration data and retained only as long as necessary for moderation purposes.

7. Data Sharing

Vanguard does not sell, rent, trade, or share data with third parties.

Data processed by Vanguard is accessible only to:

  • The bot itself
  • Authorized server moderators via standard Discord interfaces

8. Legal Basis

Data is processed on the basis of legitimate interest for server moderation and community safety, as initiated by server administrators who choose to invite Vanguard to their servers.

9. User Rights

Server administrators control how Vanguard is configured and used. Users should contact server staff regarding moderation actions or data concerns.

10. Changes to This Policy

This Privacy Policy may be updated to reflect changes in Vanguard’s functionality or legal requirements. Continued use of Vanguard constitutes acceptance of the updated policy.

11. Contact

For privacy-related questions or concerns, please contact the bot developer through the official support channel or repository associated with Vanguard.
